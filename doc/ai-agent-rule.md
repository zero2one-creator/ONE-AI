## cursor 生成本项目时代码需要遵循以下规则

1. 使用 naive ui 作为组件库
2. 使用 vue-router 作为路由管理库
3. 使用 pinia 作为状态管理库
4. 使用 scss 作为 css 库
5. 使用 unocss 进行布局，要求使用 @apply 嵌入
6. 使用 lodash 作为工具库
7. 使用 dayjs 作为时间处理工具库
8. 使用成熟的组件代替从 0 开发
9. 安装包进行询问
10. 页面的不同单元尽量拆分组件
11. 使用 yarn
12. 以前端专家的说进行代码编写

## cursor 生成新文件的位置要遵循一下规则

1. 常量或者一些不更改的状态放到 src/const 文件加下，可以用文件名区分不同类型的常量。
2. 公共组件放置到 components 下
